Estrategia Global Incom 2025 Alerta: Mitigación de Brecha Expirada (Kaspersky)

Arquitectura de defensa y continuidad operativa

Protección de servicios corporativos y gestión de movilidad segura bajo un esquema de respuesta gestionada.

Estrategia de Resiliencia Mexicana

La propuesta para Mexicana integra tres capas críticas de ciberseguridad: **Respuesta (Sophos)**, **Operación (ManageEngine)** y **Auditoría (Tenable)**. Debido al vencimiento de las licencias de Kaspersky el pasado 21 de diciembre, el enfoque inicial es la recuperación inmediata de la seguridad en el punto final (Endpoint), seguida por la implementación de una inteligencia centralizada para el control de activos y logs corporativos.

1. Fase de Mitigación

Despliegue de Sophos MDR para eliminar el riesgo de estaciones desprotegidas.

2. Fase de Visibilidad

Centralización de salud de red y logs de AD mediante ManageEngine.

3. Fase de Cumplimiento

Validación continua de brechas mediante la sinergia con Tenable.

Sophos MDR Complete: Blindaje Inmediato

Servicio de respuesta gestionada ante amenazas para la flota corporativa y de movilidad.

Bondades y Características

  • ✔
    Monitoreo Humano 24/7: Un equipo de expertos interviene activamente si se detecta un ataque a medianoche.
  • ✔
    Tecnología XDR: Correlaciona datos de red y endpoints para detener movimientos laterales.
  • ✔
    Sophos Mobile (MDM): Control absoluto de los iPads operativos, garantizando aplicaciones seguras y borrado remoto.

Requerimientos para Mexicana

  • Acceso a dominios *.sophos.com en el firewall corporativo.
  • Equipos Windows: Conexión a AD para despliegue vía GPO.
  • Equipos macOS: Credenciales administrativas para habilitar permisos de "Acceso Total al Disco".

Plan de Implementación (Fase 1)

1

Limpieza de Antivirus Previo: Desinstalación controlada de Kaspersky desde su consola central para evitar conflictos de kernel.

2

Reinicio de Estaciones: Paso crítico para liberar controladores de sistema antes de instalar el agente de Sophos.

3

Despliegue Masivo: Ejecución de instaladores MSI vía GPO para estaciones Windows.

4

Configuración MDM: Enrolamiento de iPads mediante perfiles de configuración para el personal operativo.

ManageEngine Suite: El Cerebro Operativo

Gestión de infraestructura, SIEM y remediación de vulnerabilidades.

Log360 (SIEM)

"Auditoría Forense en Tiempo Real"

Monitorea cada cambio en el AD de Mexicana. Detecta ataques de fuerza bruta y exfiltración de datos analizando comportamientos (UEBA).

Bondad: Cumplimiento normativo automatizado.

OpManager Plus

"Salud de Infraestructura 24/7"

Evita caídas de servicios corporativos monitoreando switches, routers y servidores. Alertas proactivas ante saturación de enlaces.

Bondad: Reducción de tiempos de inactividad técnica.

Vulnerability Manager

"Higiene Digital y Parchado"

Automatiza la instalación de parches de seguridad de Windows y 350+ aplicaciones de terceros. Cierra puertas antes de que sean explotadas.

Bondad: Automatización de tareas rutinarias de TI.

Plan de Implementación (Fase 2)

  • 1. Configuración del Core: Instalación de consolas en servidores virtuales y bases de datos PostgreSQL integradas.
  • 2. Mapeo de Red: Descubrimiento de activos IP en los segmentos de oficina de Mexicana.
  • 3. Integración AD: Conexión con los Controladores de Dominio para auditoría de logs.
  • 4. Dashboarding: Creación de vistas personalizadas para el monitoreo en tiempo real.

Requerimientos de Infraestructura

Nota: Los servidores pueden ser 100% VIRTUALES (VMware, Hyper-V, etc.).

  • 2 Servidores Windows Server 2019+ Mín. 16GB RAM / 8 vCPUs
  • Almacenamiento SSD 500 GB Mín.
  • Privilegios Domain Admin para despliegue

Sinergia Estratégica con Tenable

Convivencia de Alta Fidelidad

Mexicana ya posee **Tenable**, el estándar de oro en descubrimiento de brechas. En nuestro ecosistema, Tenable actúa como el **Auditor Externo** que valida la efectividad de las herramientas de ManageEngine.

Flujo de Trabajo:

"Tenable diagnostica la enfermedad -> Vulnerability Manager Plus aplica la cura -> Tenable verifica la sanación".

Auditoría vs Remediación

Tenable identifica hallazgos de configuración profunda. ManageEngine ejecuta la solución técnica de forma masiva en segundos.

Alimentación del SIEM

Los escaneos de Tenable alimentan a Log360 para tener un mapa de riesgo real basado en activos vivos de Mexicana.
